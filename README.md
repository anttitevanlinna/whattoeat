# whattoeat app

This one is a simple food suggestion app that runs on node.

## Author

anttitevanlinna (find me on twitter, if needed)

## Running Locally

node index.js


Your app should now be running on [localhost:8080](http://localhost:8080/).

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
